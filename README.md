# node.js란

자바스크립트는 웹브라우저에 내장은 자바스트립트 엔진으로 실행 됩니다 그래서 자바스트립트를 실행하는 웹브라이저를 자바이스트립트
의 구동 환경이라는 뜻에서 자바스크립트 런타임 이라고도 표현 합니다 

한마디로 node.js는  또 하나의 자바스크립트 런타임입니다 따라서 node.js는 웹브라우저 외에서도 자바스크립트를 실행시켜줍니다.
